# Setting Up Microsoft Clarity

Follow these steps to integrate Microsoft Clarity.

## Step 1: Create a Microsoft Clarity Account

1. Go to the [Microsoft Clarity](https://clarity.microsoft.com/) website.
2. Sign in with your Microsoft account and create a new project for your website.
3. Add your website details, and you'll receive a **Clarity Project ID**.
